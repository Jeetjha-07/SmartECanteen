# App Logo Assets

## Instructions
1. **Upload your logo image** as `logo.png` in this folder
2. Recommended specifications:
   - **Format**: PNG with transparent background
   - **Size**: 512x512 pixels (minimum)
   - **File name**: Must be `logo.png`

## Current Setup
✅ Folder configured in `pubspec.yaml`
✅ Splash screen ready to display the logo
✅ Ready for deployment

Just add your `logo.png` file here and run `flutter pub get`
